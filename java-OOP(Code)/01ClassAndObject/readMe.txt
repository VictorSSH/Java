 02.对象是类的实例（具体）

 04.每个对象都是独一无二的

面向对象的优势：
01.与我们的思维习惯一致！
02.提高了程序的可重用性！
03.信息隐藏，提高程序的可维护性和安全性！


面向对象的三大特性：
01.封装
02.继承
03.多态


访问权限的作用域：
  01.类
  02.方法
  03.属性

访问权限访问修饰符：
public        在任何包中的类 都能访问
protected     只能在子类中访问，不管同不同包！
private       仅限于本类中访问
default (底层没有  ，不能显式的写入)

以属性举例：
  01.如果属性
     int age;前面没有加访问修饰符！
     默认  只能在 本包中的其他类中访问此属性！
  
  02.如果是  private  int age;
          仅限于本类中访问！
     
  03.如果是  protected  int age;
         只能在子类中访问，不管同不同包！
         
  04.如果是  public  int age;
         在任何包中的类 都能访问
